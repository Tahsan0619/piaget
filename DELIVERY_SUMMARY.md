## 🚀 Quick Start

### Step 1: Get API Key (2 minutes)
```
1. Go to https://console.groq.com

4. Copy the key
```

### Step 2: Configure (1 minute)

### Step 3: Run (2 minutes)
```bash
flutter pub get
```
